# FilmFusion

# About FilmFusion
FilmFusion let you to learn about movies. There you can search any movies you want. By  clicking the movie you can learn about it. It's imdb,lengt,release date and short description about it.

## Built In
React Native
JavaSCript
TypeScript
ExpresJs
Mongodb
